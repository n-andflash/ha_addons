## 소개

* 삼성SDS 월패드를 사용하는 집에서, RS485를 이용해 여러 장치들을 제어할 수 있는 애드온입니다.
* 이 경로는 특정 사용자와 동작 여부를 확인하기 위해서만 임시로 사용하는 버전입니다.
  이외의 경우에는 [정식 버전](../sds_wallpad)을 사용해 주세요.
